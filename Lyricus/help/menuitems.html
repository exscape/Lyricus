Lyricus menu items

This help page discusses all the custom menu items in Lyricus (i.e. menu items such as Print or Find will not be mentioned in detail.

Edit > "Enter edit mode"

Enables edit mode, where you can edit the displayed lyrics. To save your changes, press ⌘S or "Save lyrics" (see below).
If you want to discard your changes, leave edit move without saving and answer "Don't save" when asked.
See editing for more information.

Edit > "Save lyrics"

Saves the lyrics that are currently displayed to the track named in the window title.
Note that Lyricus will save downloaded automatically; you only need to save if you've made changes in edit mode (see above).

Edit > "Save displayed lyrics to current track"

Unlike the standard save function above, this saves the displayed lyrics to the track that is currently playing, ignoring mismatches.
This can be useful, for instance, when searching for lyrics for cover songs:

  1. Play a cover track in iTunes; let's say that no lyrics are found for the cover artist title
  2. Search manually for the original artist instead (using ⌘N or Lyric > "Search for lyrics...")
  3. If found, save the lyrics to the still-playing cover track using Edit > "Save displayed lyrics to current track".

Lyric > "Search for lyrics..."

Search for song lyrics manually, instead of getting the artist and song name from iTunes.
Note that Lyricus will update the lyric display if iTunes is playing music and the currently playing song changes.

Lyric > "Batch downloader..."

Opens the batch downloader, which downloads lyrics for entire playlists at once.

Lyric > "Go to songmeanings page"

Opens the lyric and comment page (if any) on songmeanings.net.
Songmeanings is a large lyric community which offers users a way to comment on song themes and meanings.

Lyric > "Reverse lyric search..."

Opens the reverse lyric search window, which allows you to find songs based on words or phrases in their lyrics.

Lyric > "Update reverse lyric search index"

Updates the search index used for the reverse lyric search window.
Unless you've set the preference not to ask, Lyricus will ask you to update the index whenever it is more than one week old.
